# Historical Map of Rome

An interactive 3D historical map built with Three.js, showing Rome's terrain across 5 eras — from Ancient Rome (753 BC) through Modern Rome (2026). Zoom, orbit, and cycle through time periods to see the city's topography and landmarks evolve.

## Quick Start

```bash
npm install
npm run seed   # populate SQLite database with Rome test data
npm run dev    # starts Express API + Vite dev server
```

Open http://localhost:5173 in your browser.

## Controls

- **Mouse drag** — Pan the map
- **Scroll** — Zoom in/out
- **Right-click drag** — Rotate/orbit
- **Arrow keys** or **buttons** — Cycle through historical eras
- **Hover** near landmarks — See descriptions

## Tech Stack

- **Three.js** — 3D terrain rendering with per-vertex coloring
- **Vite** — Frontend dev server and bundler
- **Express** — Backend API server
- **better-sqlite3** — SQLite database for terrain and landmark data
- **MapControls** — Intuitive map-style navigation

## Project Structure

```
src/            Three.js frontend (main scene, terrain, UI, era manager)
server/         Express API + SQLite database logic
db/             SQLite database file (generated by seed)
.github/copilot-instructions/  Three.js skill files for VS Code Copilot
```

## Eras

1. **Ancient Rome** (753 BC – 476 AD)
2. **Medieval Rome** (476 – 1400)
3. **Renaissance** (1400 – 1600)
4. **Baroque Rome** (1600 – 1800)
5. **Modern Rome** (1800 – 2026)
